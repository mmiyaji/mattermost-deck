# Privacy Policy

Mattermost Deck stores extension settings locally in the browser.

- Stored locally: target Mattermost server URL, optional team slug, theme, language, polling interval, detailed guard settings, and optional PAT.
- PAT storage: the token is stored with client-side encryption that can be decrypted by the extension on the same client. This is not a complete security boundary.
- Network access: the extension sends requests only to the configured Mattermost server.
- No external analytics: this project does not include external analytics or telemetry by default.

Users are responsible for choosing appropriately scoped Mattermost tokens.
